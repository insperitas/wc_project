Frontend apps (customer and cleaner) - quickstart

Each app is a minimal Vite + React scaffold. To run one locally:

1. cd into the app folder, e.g. `frontend/customer`
2. `npm install`
3. `npm run dev`

The apps call the backend at `http://127.0.0.1:8000` for quick prototyping.

They are small starters intended for local development and iteration.
